关于网站

平凡派的名字来源于一本只办过三期的小小的叫做《平凡派》的文学杂志，第一次出现在2013年的苏北小县城一所高中里。当时的由一群有想法的年轻人（姚导）发起、制作、编辑，不过打印发行了少量，丝毫不意外地这个杂志很快就消失了。

我们都知道这个杂志是失败的，是幼稚的，还有毫无现实意义的，但现在无论是这本杂志的作者还是读者都会偶尔回忆当时纯粹的理想、信念。虽然《平凡派》存在时间并不长，但它对于一些人来说是意义非凡的。这些年走过来，在人生成长道路和选择上，我和他们一样依旧保持真诚善良，希望有一天会在人群中闪着光芒…

本网站会依旧秉承信念，运营和维护一些有意义的、有价值的表达，希望平凡派在我这里能够继续下去，献给更多的人。

本网站发布的文章既有原创的也有转载的，可能有部分文章转载未获得作者授权，但是我会遵守 CC 协议。

本站架构

网站采用 Hugo 发布编辑，照猫画虎抄袭了谢益辉博士的网站，所有页面都基于 Markdown 创建，源文件可以在谢益辉博士的 Github 上找到。选择这个模板主要原因是可以优雅的浏览文章，突出内容的重要性，而不是网站的动画或是技术。此外谢益辉博士加入了宋体样式，十分适合深度的阅读，感谢他的贡献和制作。对于一个无任何代码经验的人来说，暂时没有心思学习代码的敲打，本网站还有很多不足，但是我没有技术再次进行修改和再编辑，尽管有些功能未能实现，目前看来显然实现了程序员最基础的信条——已经可以“跑了”，便不再折腾，请见谅。
